Absorb is a Gameboy Color skin for the iOS emulator Delta. It supports landscape and portrait orientation and maximizes screen size.

## Currently Supports
- [ ] iPhone with Face ID (10 to 15)
    - [ ] Landscape
    - [ ] Portrait
